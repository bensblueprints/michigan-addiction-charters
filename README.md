# Michigan Addiction Charters — Premium Rebuild

Premium redesign of michiganaddictionwi.com for Captain Shawn's sport fishing charters out of Algoma, WI.

## Stack
- Next.js 14 (App Router)
- Tailwind CSS
- Framer Motion
- Cormorant + Montserrat (Google Fonts)

## Dev
```bash
npm install
npm run dev
```

## Build / Start
```bash
npm run build
npm run start
```

## Deployment
Deployed on Coolify (Contabo VPS) via Nixpacks. See `nixpacks.toml`.

## Design System
Deep Lake Cinematic — midnight + aqua + sunrise gold. Liquid-glass premium style with Cormorant display type.
